⚠️ PROBLEMA IMPORTANTE QUE RESOLVEMOS
Supabase retornando relacionamento inconsistente

Erro:

Property 'nome' does not exist...
Solução aplicada
Tipo:
type RelacaoNome = { nome: string } | { nome: string }[] | null
Helper:
function getNomeRelacao(relacao?: RelacaoNome) {
  if (!relacao) return ''
  if (Array.isArray(relacao)) return relacao[0]?.nome || ''
  return relacao.nome || ''
}
